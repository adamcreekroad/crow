# Hello World

```rb
p "Hello World"
```

```js
console.log("Hello World");
```

# String assignment

```rb
text = "Hello World"
p text
```

```js
const text = "Hello World";
console.log(text);
```

# String assignment (var)

```rb
text = "Hello"
text += " "
text += "World"
p text
```

```js
let text = "Hello";
text = text + " ";
text = text + "World";
console.log(text);
```

# Concatenating strings

```rb
p "Hello" + "World"
```

```js
console.log("Hello" + "World");
```

# Interpolated string

```rb
who = "World"
p "Hello #{who}"
```

```js
const who = "World";
console.log(`Hello ${who}`);
```

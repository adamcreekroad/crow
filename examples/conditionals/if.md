# If statements

```crystal
if true
  p "yes"
end
```

```js
if (true) {
  console.log("yes");
}
```

# If and else

```crystal
if true
  p "yes"
else
  p "no"
end
```

```js
if (true) {
  console.log("yes");
} else {
  console.log("no");
}
```

# Elsif and else

```crystal
if true
  p "yes"
elsif false
  p "maybe"
else
  p "no"
end
```

```js
if (true) {
  console.log("yes");
} else if (false) {
  console.log("maybe");
} else {
  console.log("no");
}
```
